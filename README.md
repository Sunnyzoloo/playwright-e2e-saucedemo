# Playwright E2E Automation – SauceDemo

End-to-end test automation project built with Playwright and TypeScript to validate core e-commerce user flows.

## Tech Stack

- Playwright
- TypeScript
- Node.js

## Test Coverage

- Login scenarios (valid user, invalid password, locked-out user) using data-driven tests
- Inventory functionality: adding a product to the cart
- Cart functionality: verifying added products

## How to Run the Tests

```bash
npm install
npx playwright test
```
